🚀 **SAP ABAP RAP Project | Travel Management Application on SAP BTP (Steampunk)**

Successfully designed and developed a complete **Travel Management Application** using the **ABAP RESTful Application Programming Model (RAP)** on **SAP BTP ABAP Environment (Steampunk)**.

### Key Highlights

🔹 Built a transactional RAP application with a three-level business object:

* **Travel**
* **Booking**
* **Booking Supplement**

🔹 Implemented:

* Managed RAP Business Objects
* Early Numbering for Travel, Booking, and Booking Supplement entities
* Root and child compositions with CBA (Create by Association)
* EML (Entity Manipulation Language) for transactional processing
* Factory Action to **Copy Travel** along with all associated Bookings and Booking Supplements
* Optimistic locking using **ETags**
* Instance and global authorization framework
* CDS View Entities with associations, compositions, and transactional behavior definitions
* Fiori Elements Object Page with complete CRUD operations
